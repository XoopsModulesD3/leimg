Leimg
=====
Leimg is image management module for other modules.
Module developers can use this module for handle images uploaded through form.
This module save the uploaded file at the proper directory(html/uploads/), and make thumbnails as to settings of this module.

Environment
-----------
XOOPS Cube Legacy 2.2 or later.

Setup
-----
Add thumbnail settings for each client module.

Main Feature
------------
- Save uploaded files.
- Make thumbnails.
- Set thumbnail size and number.
- Upload blank image.
